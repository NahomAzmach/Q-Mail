Security Information

Important Security Notes

  • App Passwords: For services with 2-factor authentication (like Gmail), use an app password instead of your regular password.
  • No Stored Credentials: Your email credentials are not stored persistently by this application.
  • Secure Connection: All connections to email providers use SSL encryption.

How to Get an App Password

  1. For Gmail:
    • Go to your Google Account → Security → 2-Step Verification → App passwords
    • Create a new app password specifically for this app
  2. For Outlook/Hotmail:
    • Go to your Microsoft account → Security → Advanced security options → App passwords
  3. For Yahoo:
    • Go to Account security → Generate app password
